## Configuration

```
topic_listen: irur/tele/RESULT
topic_send:
- irur/cmnd/IRsend
```

`topic_listen` (str)  
MQTT topic for receiving IR codes

`topic_send` (list)  
MQTT topic(s) for sending IR codes
